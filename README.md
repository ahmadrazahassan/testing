# Gooomly

React (Vite) marketplace UI — components and pages in JSX, logic in JavaScript modules under `src/`.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

## Production build

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```

Static files (images, etc.) live in `public/` and are served from the site root (e.g. `/assets/images/...`).
